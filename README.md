# GymLog - Health Calculator

**Your Personal Fitness Companion**

GymLog is a comprehensive health and fitness calculator built with Flutter. It helps users track daily proteins, calories, water intake, and sleep hours, providing personalized recommendations based on BMR (Basal Metabolic Rate) and TDEE (Total Daily Energy Expenditure) calculations.

## üöÄ Features

-   **Smart Calculators**:
    -   **BMR & TDEE**: Accurate calorie targets based on the Mifflin-St Jeor equation.
    -   **Protein & Macros**: Personalized protein, carbohydrate, and fat intake recommendations.
    -   **Water Intake**: Hydration goals based on body weight and activity level.
    -   **Sleep Tracker**: Recovery tracking and sleep quality monitoring.

-   **Daily Logging**:
    -   Easy-to-use sliders for logging daily intake.
    -   Visual progress rings and streak tracking.
    -   Quick-add buttons for common values.

-   **User Profile**:
    -   Customizable fitness goals (Lose Weight, Maintain, Gain Muscle).
    -   Support for both **Metric (kg/cm)** and **Imperial (lbs/in)** unit systems.
    -   Local data storage for complete privacy.

-   **Premium Design**:
    -   Sleek Dark Theme with "Clean Green" accents.
    -   Smooth animations and intuitive UI.

## üèÅ Getting Started

### Prerequisites
-   Flutter SDK (^3.6.0)
-   Dart SDK (^3.6.0)

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/abdellatifaitbaali/GymLog.git
    cd GymLog
    ```

2.  **Install dependencies**
    ```bash
    flutter pub get
    ```

3.  **Run the app**
    ```bash
    flutter run
    ```

4.  **Build Release APK**
    ```bash
    flutter build apk --release
    ```
